Clingy-Footer
Clingy-Footer is for fluid height footers. If you can fix your footer's height, you should use one of the many CSS only solutions available. (I recommend this one).

Basic Use

Have a look at the source of this page. All you have to do is call the function (if you use the default selectors):

$(document).ready(function(){
	$(document).clingyFooter();
});

Overriding Default Selectors

You can also easily pass in the selectors for the body of the page (pageContentSelector), the footer (footerSelector) and the wrapper around the two (pageSelector) to override the default IDs shown in this example:

$(document).ready(function(){
	$(document).clingyFooter({
		pageSelector : '#page',
		pageContentSelector : '#page-body',
		footerSelector: "#page-footer"
	});
});

Supported Usage

Clingy-Footer is supported on jQuery v2.0.3 (and therefore should work in all browsers that jQuery supports) but should work on most older versions as it's pretty simple. I do not currently, nor will I ever, look at this in IE8 or below. Life is too short!
