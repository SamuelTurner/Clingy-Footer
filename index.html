Clingy-Footer
Clingy-Footer is for fluid height footers. If you can fix your footer's height, you should use one of the many CSS only solutions available. (I recommend this one).

Supported Usage

Clingy-Footer runs on jQuery v2.0.3 (so should work in all browsers jQuery supports) but should work on most older versions as it's pretty simple.

View the code on Github

Basic Use

All you have to do is call the function (if you use the default selectors).

$(document).ready(function(){
  $(document).clingyFooter();
});

Overriding Default Selectors

You can easily pass in the selectors for the body of the page (pageContentSelector), the footer (footerSelector) and the wrapper around the two (pageSelector) to override the defaults.

$(document).ready(function(){
  $(document).clingyFooter({
    pageSelector: '#page',
    pageContentSelector: '#page-body',
    footerSelector: "#page-footer"
  });
});
